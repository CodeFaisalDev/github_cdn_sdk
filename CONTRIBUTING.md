# Contributing to github-cdn-sdk

First off, thank you for considering contributing to `github-cdn-sdk`! It's people like you who make it such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub issue tracker](https://github.com/CodeFaisalDev/github_cdn_sdk/issues).
- Describe the bug and include steps to reproduce it.

### Suggesting Enhancements
- Open a new issue with the tag `enhancement`.
- Explain how the feature would work and why it's useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (`npm test`).
4. Make sure your code lints.
5. Issue that pull request!

## Style Guide
- Use TypeScript.
- Follow the existing code style (clean, modular, well-documented).
